## 1. 产品概述
「冰路先锋」是一款基于浏览器的冷饮箱补货调度休闲游戏。玩家扮演冷饮物流调度员，根据库存、路程消耗和随机事件，规划最优补货路线，优先处理紧急需求。
- 核心玩法：通过拖拽排序路线步骤、切换补货优先级、执行补货操作来推进回合
- 目标用户：休闲游戏爱好者、策略游戏玩家
- 产品价值：轻量级调度模拟，结合路线规划与应急决策，提供有趣的策略挑战

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 体验完整游戏、保存成绩到本地 |

### 2.2 功能模块
1. **主菜单页**：开始游戏、选择关卡、教程、查看历史成绩
2. **游戏主界面**：地图展示补货点、路线规划区、库存面板、事件通知、操作控制
3. **结算界面**：分数、缺口处理效率、路线冗余度、异常响应速度、剩余库存
4. **暂停/教程弹窗**：暂停游戏、查看操作说明

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主菜单页 | 关卡选择 | 展示所有可用关卡，显示通关星级 |
| 主菜单页 | 历史成绩 | 展示本地存储的各关卡最佳成绩 |
| 游戏主界面 | 补货地图 | 可视化展示仓库与各补货点位置、连接通道 |
| 游戏主界面 | 路线队列 | 拖拽排序补货步骤，显示预计消耗 |
| 游戏主界面 | 库存面板 | 展示当前总库存、各类冷饮数量 |
| 游戏主界面 | 事件面板 | 随机事件弹出与处理选项 |
| 游戏主界面 | 操作栏 | 执行补货、暂停、重置本关 |
| 结算界面 | 数据统计 | 多维度结算数据展示与评级 |

## 3. 核心流程
玩家从主菜单选择关卡进入游戏，查看补货点需求与当前库存，规划补货路线（可拖拽排序），应对随机事件，点击执行补货操作，消耗库存与时间，完成所有补货后进入结算界面查看成绩。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：清凉蓝色系（#0EA5E9 天蓝 / #0284C7 深蓝 / #7DD3FC 浅蓝），辅以冰白色背景
- **强调色**：橙红色（#F97316）用于事件警告与紧急标记
- **成功色**：翠绿色（#10B981）用于补货完成与正向事件
- **按钮风格**：圆角胶囊按钮，悬停上浮微动画
- **字体**：标题用圆润显示字体，正文用清晰无衬线字体
- **布局风格**：卡片式分区布局，地图居中，两侧信息面板
- **图标风格**：Emoji + 简洁矢量图标（🧊 🚚 ❄️ ⚠️ 🏆）

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主菜单页 | 关卡卡片 | 蓝白渐变卡片，悬浮微动画，星级显示 |
| 游戏主界面 | 补货地图 | 节点连接线，动态高亮当前路线，补货点显示需求进度条 |
| 游戏主界面 | 路线队列 | 可拖拽列表项，序号徽章，预计消耗提示 |
| 游戏主界面 | 事件弹窗 | 半透明遮罩，卡片式事件说明，操作按钮组 |
| 结算界面 | 数据面板 | 进度条形式展示各维度指标，最终评级徽章 |

### 4.3 响应式
- 桌面端优先设计（≥1024px），地图居中两侧面板
- 平板端（768-1024px）信息面板堆叠在地图下方
- 移动端（<768px）简化地图显示，垂直堆叠布局
- 触摸设备支持长按拖拽排序

### 4.4 视觉动效
- 页面加载：错落淡入动画
- 补货执行：卡车图标沿路线移动动画
- 事件触发：警示脉冲动画
- 拖拽排序：被拖拽项半透明放大，目标位置指示线
- 结算展示：数字滚动累加效果
